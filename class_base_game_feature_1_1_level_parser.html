BaseGameFeature::LevelParser Class Reference

#include <levelparser.h>

Detailed Description

Base class for level parsing.

Entity format is defined as:

{
"level": {
"version": "100"
"entities": {
"[GUID]": {
"name": String,
"components": {
"ComponentName1": VALUE,
"ComponentName2": {
"FieldName1": VALUE,
"FieldName2": VALUE
}
},
},
"[GUID]": {
...
}
}
}
}
